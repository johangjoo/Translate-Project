# 🎯 여기서 시작하세요!

## 📁 프로젝트 구조 확인

```
your_project/
├── LoraScript/              # ⭐ 이 폴더
│   ├── setup.sh
│   ├── prepare_dataset.py
│   ├── train_qwen_lora.py
│   ├── inference_lora.py
│   └── ...
└── json_data/               # 여기에 데이터 배치
    ├── ko_to_ja/
    └── ja_to_ko/
```

---

## ⚡ 3단계로 시작하기

### 1단계: 환경 설정 (5분) ⚙️

```bash
cd LoraScript
bash setup.sh
```

✅ Python, GPU, Unsloth 자동 설치  
✅ 폴더 구조 자동 생성

**문제 발생 시:** [STEP1_환경설정.md](STEP1_환경설정.md) 참고

---

### 2단계: 데이터 준비 (10분) 📊

```bash
# 1. 프로젝트 루트로 이동
cd ..

# 2. 숫자 폴더들 배치
mv 801 802 803 json_data/ko_to_ja/
mv 901 902 json_data/ja_to_ko/

# 3. 전처리 실행
cd LoraScript
python prepare_dataset.py
```

✅ 재귀 탐색으로 모든 JSON 파일 자동 탐색  
✅ train.jsonl, validation.jsonl 자동 생성

**문제 발생 시:** [STEP2_데이터준비.md](STEP2_데이터준비.md) 참고

---

### 3단계: 학습 실행 (1~10시간) 🔥

```bash
python train_qwen_lora.py
```

✅ 12GB VRAM 최적화  
✅ 자동 체크포인트 저장  
✅ TensorBoard 로깅

**문제 발생 시:** [STEP3_학습실행.md](STEP3_학습실행.md) 참고

---

## 🎉 학습 완료 후

python inference_lora.py
